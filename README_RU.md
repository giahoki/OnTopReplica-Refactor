# OnTopReplica Refactor

**Клон любого окна в реальном времени с режимом «всегда поверх остальных» для Windows Vista, 7, 8, 10 и 11.**

Это простое утилитарное приложение по умолчанию показывает пустое окно, которое всегда остаётся поверх остальных. Пользователи могут выбрать любое другое окно системы, чтобы получить его всегда актуальную копию поверх всех окон. Очень полезно для мониторинга фоновых процессов, работы со сложными многооконными играми или инструментами, просмотра YouTube-видео во время работы и т.д.

Это улучшенный форк оригинального [OnTopReplica от LorenzCK](https://github.com/LorenzCK/OnTopReplica) с миграцией на .NET 4.8, поддержкой Windows 11, горячими клавишами, выбором языка, расширенными полноэкранными режимами и многим другим.

## 📢 Возможности

### Основные функции
- **Клонирование любого окна** с режимом *всегда поверх остальных*
- **Выбор подобласти** клонированного окна:
  - Можно сохранить для будущего использования
  - Можно использовать относительные координаты от границ целевого окна
- **Автоматическое изменение размера** — вписать в оригинальное окно, половина, четверть и полноэкранный режим
- **Фиксация позиции** в любом углу экрана
- **Регулировка прозрачности** для прозрачных оверлеев

### Взаимодействие
- **«Передача кликов»** — взаимодействие с клонированным окном напрямую
- **«Пропуск кликов»** — реплика игнорирует любые действия мыши (превращает OnTopReplica в оверлей при частичной прозрачности)
- **Режим «Групповое переключение»** — автоматическое переключение между группой окон во время использования

### Новое в Refactor ✨
- **️ Глобальные горячие клавиши**
  - Настраиваемая клавиша для показа/скрытия OnTopReplica
  - Настраиваемая клавиша для клонирования текущего активного окна
  - Управление горячими клавишами в панели настроек

- **🌍 Мультиязычная поддержка**
  - Выбор из 12+ языков интерфейса с иконками флагов
  - English, Čeština, Dansk, Deutsch, Español, Italiano, Polski, Русский, 简体中文, 繁體中文, Português, 日本語
  - Автоматическое определение системного языка по умолчанию

- **️ Расширенные полноэкранные режимы**
  - **Стандартный** — заполняет рабочую область (исключая панель задач)
  - **Полный экран** — покрывает весь экран, включая панель задач
  - **Все экраны** — растягивается на все подключённые мониторы

- ** Технические улучшения**
  - Миграция на **.NET Framework 4.8**
  - Обновление до последней версии библиотеки **WindowsFormsAero**
  - Поддержка **Windows 11**
  - Удалена система автообновления для более стабильных сборок
  - Улучшенное позиционирование и размер форм (без белых границ)
  - MSI-установщик для простого развёртывания

## Использование

### Базовое
1. Запустите OnTopReplica — появится пустое окно поверх всех окон
2. Нажмите правую кнопку мыши для открытия контекстного меню
3. Выберите **Select Window** и кликните на окно, которое хотите клонировать
4. Реплика появится в окне OnTopReplica

### Продвинутое
- **Выбор области** — нажмите правую кнопку и выберите область клонированного окна
- **Полноэкранный режим** — переключайтесь между режимами для immersive просмотра
- **Прозрачность** — настройте прозрачность для использования в качестве оверлея
- **Горячие клавиши** — настройте глобальные сочетания в настройках для быстрого доступа
- **Язык** — измените язык интерфейса в настройках
